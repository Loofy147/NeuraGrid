# NeuroGrid API Demo

This is a technical demo showcasing the functionality of the NeuroGrid API. The demo displays a code example for fetching real-time neural data and a simulated API response.

## Getting Started

To run the demo locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/neurogrid-demo.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd neurogrid-demo
    ```
3.  **Install the dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm start
    ```
    The demo will be available at [http://localhost:3000](http://localhost:3000).

## Project Structure

The project is a standard React application with the following structure:

-   `public/`: Contains the `index.html` file and other static assets.
-   `src/`: Contains the React components:
    -   `App.js`: The main component that renders the other components.
    -   `Header.js`: The header component for the demo.
    -   `ApiCode.js`: The component that displays the API code example.
    -   `ApiResponse.js`: The component that displays the simulated API response.
